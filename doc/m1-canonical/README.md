# M1 Canonical Scope

`doc/m1-canonical/` contains BEAR workflow guidance that may be copied into demo repos.

In scope here:
- `BEAR_PRIMER.md`
- `AGENTS.md`
- `BEAR_AGENT.md`
- `WORKFLOW.md`

Out of scope here:
- demo-domain app specs
- demo block index content

Those artifacts are owned directly by `bear-account-demo`.
